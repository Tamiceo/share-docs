Sales Playbook

세일즈 리드 확보를 위한
행사/세미나 프로세스 가이드

세미나·행사를 통해 세일즈 파이프라인에 기여하는 MQL을 체계적으로 확보한다
💡 좋은 행사는 교육처럼 보이지만, 실제로는 세일즈 행사다
1
행사 준비 — 목표와 전략 싱크
브랜딩 레이어
사고 리더십 확보, 인지도 확산
KPI: 참석자 수, 소셜 언급량
세일즈 레이어
파이프라인 유입 리드 확보
KPI: MQL 수, 미팅 예약 수
어필 포인트 예시
브랜딩 측면 "데이터 분석가들의 AI 활용 전략" — Thought Leadership 포지셔닝
세일즈 측면 고객사에 AI 피처 어필 + 네트워킹을 통한 리드 확보 및 관계 강화
타겟 설정 팁: 제목 자체가 필터다. "AI를 쓰는 데이터팀 팀장"이 제목을 보고 '이거 나 얘기네'라고 느껴야 한다.
2
R&R — 협조 요청 및 역할 분담
역할 담당 핵심 책임
세일즈 세일즈팀 기존 파이프라인 개인 초대, 당일 네트워킹·고객 응대, 행사 후 핫리드 팔로업
컨설팅 컨설팅팀 행사 콘텐츠 협업(세션 기획·발표), Q&A 대응, 고객 응대
마케팅 마케팅팀 장소·스피커·참여자 섭외, 행사 홍보 (에이전시 협업 포함)
협업 마케팅+세일즈 행사 스폰서십 확보
리더십 C-레벨 전체 의견 탑업, 키노트·모두발언·패널토크 참여
3
행사 타임라인 — 소규모 세미나 기준 6주
행사 계획 준비
→
사전 홍보
→
행사 당일
→
사후 홍보 & 리드 대응
D-42 · 6주 전
기획 확정
  • 목표·KPI·타겟 ICP 확정
  • 연사 섭외, 콘텐츠 아웃라인
  • 등록 시스템(Google Forms + 랜딩) 세팅
D-28~14 · 4~2주 전
사전 홍보 — 티저 바이럴 모객
  • 이메일 캠페인 1차 발송 (기존 DB 대상)
  • 세일즈팀 → 타겟 계정 개인 초대 시작
  • 연사 소개 콘텐츠 발행 (신뢰도 빌드업)
  • 등록자 사전 설문 발송 (고통점 파악)
D-3 · 3일 전
확정 준비
  • 세일즈팀에 등록자 명단 공유 → 사전 리서치 시작
  • 리마인더 이메일 발송
D-Day
행사 당일
  • 현황 스케치, 고객 보이스·반응 수집
  • 세일즈팀 집중 배치 → 네트워킹 관계 확보 (약속 잡기)
  • 미팅 희망자 즉시 예약 유도 (Google Calendar QR)
  • Q&A·설문으로 관심 신호 실시간 수집
D+1~7 · 행사 사후
사후 홍보 및 리드 대응
  • 행사 자료 세분화 → 바이럴 콘텐츠로 사후 홍보·확대
  • 전체 참석자 감사 이메일 (세션 요약 + 자료 링크)
  • 불참자 → 녹화 영상 + 별도 이메일
4
행사 리드 대응 — 팔로업 SLA (팀 내 응답 기한 약속)
🔥 핫리드
24시간
미팅 신청, 명함 교환 10분+
→ 세일즈 직접 콜·이메일
🌤 웜리드
48시간
Q&A 참여, 설문 관심 표시
→ 개인화 이메일 + 자료
🌊 콜드리드
1주 내
단순 참석, 관심 표명
→ 뉴스레터 너처링 시퀀스
💤 불참자
1주 내
등록했지만 No-show
→ 녹화/요약 발송 후 재판단
핵심: 행사 후 24시간이 골든타임. 핫리드를 빠르게 접촉할수록 미팅 예약률 3~5배 차이.
5
도구/환경 세팅 — Google Workspace + Claude Code 기반
단계 역할 도구
행사 등록·참석자 관리 신청 폼, 리마인더 자동 발송 Google Forms + Gmail 예약 발송
리드 DB 관리 참석자 정보 수집·정리·Tier 분류 Google Sheets
이메일 캠페인 홍보 이메일, 팔로업 발송 Gmail + Google Sheets 연동
팔로업 SLA 관리 기한 도래 알림, 미팔로업 체크 Sheets 조건부 서식 + 슬랙 알림
미팅 예약 핫리드 즉시 예약 유도 Google Calendar 링크 공유
설문·피드백 관심 신호 수집 → Tier 분류 근거 Google Forms
🤖 Claude Code가 해줄 수 있는 것
작업 활용 방법
Tier 분류 Sheets 데이터 붙여넣으면 관심 신호 기준으로 자동 분류
개인화 팔로업 이메일 리드별 이름·회사·관심사 입력 → 이메일 초안 일괄 생성
행사 리캡 콘텐츠 세션 메모·녹취 입력 → 블로그 리캡, SNS 카드뉴스 소재 초안
설문 분석 응답 데이터 → 고통점 패턴 분석 + 세션 콘텐츠 방향 제안
SLA 현황 리포트 Sheets 데이터 기반 팔로업 현황 요약 → 슬랙 공유용 리포트
스피커 섭외 메일 행사 목적·타겟 설명 → 연사 제안 이메일 초안 생성

용어 정리

MQL — 마케팅으로 관심을 보인 잠재고객. 세일즈 팔로업 대상 리드
ICP — 이상적 고객 프로파일 (업종·규모·직책 등)
SLA — 팀 내 대응 기한 약속. 예) 핫리드 24시간 내 연락
KPI — 핵심 성과 지표. 행사 성과를 측정하는 수치 기준
R&R — 역할과 책임 분담
CRM — 고객·리드 정보 관리 시스템
너처링 — 준비 안 된 리드에게 콘텐츠를 지속 제공해 관계 유지
ROI — 행사 비용 대비 파이프라인·매출 기여도
